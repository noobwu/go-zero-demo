项目前端是基于element admin ui来创建的，后端是基于go-zero来创建的一个前后端分离的管理系统

[如果喜欢Ant Design Pro前端版本的，可以点击查看](https://github.com/feihua/go-zero-admin-ui)

# 1.安装 node_modules:

```shell
npm install
```
# 2.启动项目

```shell
npm run dev
```
